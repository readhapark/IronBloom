How this MVP works

IronBloom focuses on three core features you can ship fast: a curated recipe library, a simple daily iron tracker, and a proof-of-concept AI chatbot grounded in known, researched facts.

1

Browse iron-rich recipes

A local JSON file powers a visually rich recipe library with ingredients, instructions, iron mg per serving, enhancers and inhibitors.

2

Log meals in a few taps

From the recipe detail page, tap Add to Today to instantly add that recipe’s iron value into localStorage and update a simple daily progress bar.

3

Chat with an iron-aware coach

A minimal chat UI sends questions to an AI API with an in-context prompt that includes your 15–20 facts about iron and your recipe list – simulating a safe, focused nutrition assistant.

Core screens in the prototype

A simple dashboard showing today’s iron total vs. a hard-coded goal (e.g., 18mg), plus a list of logged recipes.

AI Chat

Lightweight chat interface that sends questions and your iron facts to an AI API for grounded answers.
